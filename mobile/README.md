# kcart Mobile (React Native Skeleton)

This is a minimal React Native skeleton using Expo to mirror the iOS-inspired design.

Quickstart:
1. Install Expo CLI if needed:
   ```bash
   npm i -g expo
   ```
2. In this folder:
   ```bash
   npm install
   npm start
   ```
3. Open on a simulator or device via Expo Go.

Notes:
- This is a presentational skeleton; data is placeholder only.
- Styles follow the web’s glassmorphism and palette.
